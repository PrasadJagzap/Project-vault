# Travello – Travel Booking Website

## 📌 Project Overview
Travello is a simple travel booking website built using **HTML and CSS**.  
It allows users to explore destinations, view tour packages, and book trips through an online booking form.

This project demonstrates basic **front-end web development concepts** including navigation menus and forms.

---

## 🚀 Features
- Travel website layout
- Navigation bar with multiple pages
- Tour package selection
- Online booking form
- Gallery section
- Contact page
- Simple and clean design
